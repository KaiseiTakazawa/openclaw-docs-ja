# configure

# `openclaw configure`

資格情報、デバイス、およびエージェントのデフォルトを設定するためのインタラクティブプロンプトです。

注意: **Model** セクションには、`agents.defaults.models` のホワイトリスト用のマルチセレクトが含まれるようになりました（`/model` およびモデルピッカーに表示されるもの）。

ヒント: サブコマンドなしの `openclaw config` は同じウィザードを開きます。非インタラクティブな編集には `openclaw config get|set|unset` を使用してください。

関連情報:

* Gateway 設定リファレンス: [Configuration](/gateway/configuration)
* Config CLI: [Config](/cli/config)

注意事項:

* Gateway の実行場所を選択すると、常に `gateway.mode` が更新されます。それが必要なすべてであれば、他のセクションなしで「Continue」を選択できます。
* チャンネル指向のサービス（Slack/Discord/Matrix/Microsoft Teams）は、セットアップ中にチャンネル/ルームのホワイトリストを求めます。名前またはIDを入力できます; ウィザードは可能な場合、名前をIDに解決します。

## 例

```bash
openclaw configure
openclaw configure --section model --section channels
```
