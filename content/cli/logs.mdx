# logs

# `openclaw logs`

RPCを介してGatewayファイルのログを表示します（リモートモードで動作します）。

関連情報:

* ロギングの概要: [Logging](/logging)

## 例

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
openclaw logs --local-time
openclaw logs --follow --local-time
```

`--local-time`を使用して、タイムスタンプをローカルタイムゾーンで表示します。
