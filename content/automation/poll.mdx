# ポーリング
# ポーリング
## 対応チャネル
\* WhatsApp (ウェブチャネル)
\* Discord
\* MS Teams (アダプティブカード)
## CLI
```bash
# WhatsApp
openclaw message poll --target +15555550123 \
--poll-question "今日のランチは？" --poll-option "はい" --poll-option "いいえ" --poll-option "多分"
openclaw message poll --target 123456789@g.us \
--poll-question "会議の時間は？" --poll-option "午前10時" --poll-option "午後2時" --poll-option "午後4時" --poll-multi
# Discord
openclaw message poll --channel discord --target channel:123456789 \
--poll-question "おやつは？" --poll-option "ピザ" --poll-option "寿司"
openclaw message poll --channel discord --target channel:123456789 \
--poll-question "計画は？" --poll-option "A" --poll-option "B" --poll-duration-hours 48
# MS Teams
openclaw message poll --channel msteams --target conversation:19:abc@thread.tacv2 \
--poll-question "ランチは？" --poll-option "ピザ" --poll-option "寿司"
```
オプション:
\* `--channel`: `whatsapp` (デフォルト), `discord`, または `msteams`
\* `--poll-multi`: 複数のオプションを選択可能にする
\* `--poll-duration-hours`: Discord専用（省略時は24時間）
## Gateway RPC
メソッド: `poll`
パラメータ:
\* `to` (文字列, 必須)
\* `question` (文字列, 必須)
\* `options` (文字列\[], 必須)
\* `maxSelections` (数値, 任意)
\* `durationHours` (数値, 任意)
\* `channel` (文字列, 任意, デフォルト: `whatsapp`)
\* `idempotencyKey` (文字列, 必須)
## チャネルの違い
\* WhatsApp: 2-12オプション、`maxSelections`はオプション数内でなければならず、`durationHours`は無視される。
\* Discord: 2-10オプション、`durationHours`は1-768時間に制限される（デフォルトは24）。`maxSelections > 1`はマルチ選択を有効にする; Discordは厳密な選択数をサポートしていない。
\* MS Teams: アダプティブカードポール（OpenClaw管理）。ネイティブのポールAPIはなく、`durationHours`は無視される。
## エージェントツール (メッセージ)
`message`ツールを`poll`アクション（`to`, `pollQuestion`, `pollOption`, 任意の`pollMulti`, `pollDurationHours`, `channel`）で使用します。
注意: Discordには「正確にNを選択する」モードはなく、`pollMulti`はマルチ選択にマッピングされます。
Teamsのポールはアダプティブカードとしてレンダリングされ、投票を記録するためにゲートウェイがオンラインの状態である必要があります。`~/.openclaw/msteams-polls.json`に記録されます。
