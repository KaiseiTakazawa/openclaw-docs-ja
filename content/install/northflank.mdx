# Northflankにデプロイ

OpenClawをNorthflankにワンクリックテンプレートでデプロイし、ブラウザでセットアップを完了します。  
これは「サーバー上でターミナルなし」の最も簡単な方法です：NorthflankがGatewayを実行し、すべてを`/setup`ウェブウィザードを通じて設定します。

## 始め方

1. [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw)をクリックしてテンプレートを開きます。
2. まだアカウントを持っていない場合は、[Northflankにアカウントを作成](https://app.northflank.com/signup)します。
3. **Deploy OpenClaw now**をクリックします。
4. 必要な環境変数を設定します：`SETUP_PASSWORD`。
5. **Deploy stack**をクリックしてOpenClawテンプレートをビルドして実行します。
6. デプロイが完了するまで待ち、次に**View resources**をクリックします。
7. OpenClawサービスを開きます。
8. 公開されたOpenClawのURLを開き、`/setup`でセットアップを完了します。
9. `/openclaw`でControl UIを開きます。

## 取得できるもの

* ホスティングされたOpenClaw Gateway + Control UI
* `/setup`でのウェブセットアップウィザード（ターミナルコマンドなし）
* Northflank Volume（`/data`）を介した永続ストレージにより、設定/資格情報/ワークスペースが再デプロイ後も生き残ります

## セットアップフロー

1. `https://<your-northflank-domain>/setup`にアクセスし、`SETUP_PASSWORD`を入力します。
2. モデル/認証プロバイダーを選択し、キーを貼り付けます。
3. （オプション）Telegram/Discord/Slackトークンを追加します。
4. **Run setup**をクリックします。
5. `https://<your-northflank-domain>/openclaw`でControl UIを開きます。

TelegramのDMがペアリングに設定されている場合、セットアップウィザードがペアリングコードを承認できます。

## チャットトークンの取得

### Telegramボットトークン

1. Telegramで`@BotFather`にメッセージを送信します。
2. `/newbot`を実行します。
3. トークンをコピーします（`123456789:AA...`のように見えます）。
4. `/setup`に貼り付けます。

### Discordボットトークン

1. [https://discord.com/developers/applications](https://discord.com/developers/applications)にアクセスします。
2. **New Application** → 名前を選択します。
3. **Bot** → **Add Bot**を選択します。
4. Bot → Privileged Gateway Intentsの下で**MESSAGE CONTENT INTENTを有効にします**（必須。そうしないとボットが起動時にクラッシュします）。
5. **Bot Token**をコピーし、`/setup`に貼り付けます。
6. ボットをサーバーに招待します（OAuth2 URL Generator; スコープ: `bot`, `applications.commands`）。
