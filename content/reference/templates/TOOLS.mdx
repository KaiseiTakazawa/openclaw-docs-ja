# TOOLS.md テンプレート

# TOOLS.md - ローカルノート

スキルはツールが*どのように*機能するかを定義します。このファイルは*あなた*の特定の内容、つまりあなたのセットアップに特有のもののためのものです。

## ここに何を書くか

例えば以下のようなこと：

* カメラの名前と場所
* SSHホストとエイリアス
* TTSのための好みの声
* スピーカー/部屋の名前
* デバイスのニックネーム
* 環境特有の何でも

## 例

```markdown
### カメラ

- living-room → メインエリア、180°広角
- front-door → 入口、動作トリガー

### SSH

- home-server → 192.168.1.100, ユーザー: admin

### TTS

- 好みの声: "Nova"（温かみがあり、ややイギリス英語）
- デフォルトスピーカー: キッチンのHomePod
```

## なぜ分けるのか？

スキルは共有されます。あなたのセットアップはあなたのものです。それらを分けておくことで、ノートを失うことなくスキルを更新でき、インフラを漏らすことなくスキルを共有できます。
