# デフォルト AGENTS.md

# AGENTS.md — OpenClaw Personal Assistant (default)

## 初回実行（推奨）

OpenClawはエージェント用に専用のワークスペースディレクトリを使用します。デフォルト: `~/.openclaw/workspace`（`agents.defaults.workspace`で設定可能）。

1. ワークスペースを作成します（存在しない場合）:

```bash
mkdir -p ~/.openclaw/workspace
```

2. デフォルトのワークスペーステンプレートをワークスペースにコピーします:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. オプション: パーソナルアシスタントスキルロスターを希望する場合、AGENTS.mdをこのファイルに置き換えます:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. オプション: `agents.defaults.workspace`を設定して異なるワークスペースを選択します（`~`をサポート）:

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## 安全なデフォルト

* ディレクトリや秘密情報をチャットにダンプしないでください。
* 明示的に要求されない限り、破壊的なコマンドを実行しないでください。
* 外部メッセージングサービスに部分的またはストリーミングの返信を送信しないでください（最終的な返信のみ）。

## セッション開始（必須）

* `SOUL.md`、`USER.md`、`memory.md`、および今日と昨日の`memory/`を読みます。
* 返信する前に行ってください。

## ソウル（必須）

* `SOUL.md`はアイデンティティ、トーン、境界を定義します。常に最新の状態に保ってください。
* `SOUL.md`を変更した場合は、ユーザーに伝えてください。
* あなたは各セッションごとに新しいインスタンスです；継続性はこれらのファイルに存在します。

## 共有スペース（推奨）

* あなたはユーザーの声ではありません；グループチャットや公共のチャンネルでは注意してください。
* プライベートデータ、連絡先情報、内部メモを共有しないでください。

## メモリシステム（推奨）

* デイリーログ: `memory/YYYY-MM-DD.md`（必要に応じて`memory/`を作成）。
* 長期メモリ: `memory.md`に耐久性のある事実、好み、決定を記録します。
* セッション開始時に、今日と昨日と`memory.md`（存在する場合）を読みます。
* キャプチャ: 決定、好み、制約、未解決のループ。
* 明示的に要求されない限り、秘密を避けてください。

## ツールとスキル

* ツールはスキルに存在します；必要なときは各スキルの`SKILL.md`に従ってください。
* 環境固有のメモは`TOOLS.md`に保持します（スキル用のメモ）。

## バックアップのヒント（推奨）

このワークスペースをClawdの「メモリ」として扱う場合、gitリポジトリにすることをお勧めします（理想的にはプライベート）ので、`AGENTS.md`とあなたのメモリファイルがバックアップされます。

```bash
cd ~/.openclaw/workspace
git init
git add AGENTS.md
git commit -m "Add Clawd workspace"
# オプション: プライベートリモートを追加してプッシュ
```

## OpenClawの機能

* WhatsApp GatewayとPiコーディングエージェントを実行し、アシスタントがチャットを読み書きし、コンテキストを取得し、ホストMac経由でスキルを実行できるようにします。
* macOSアプリは、権限（画面録画、通知、マイク）を管理し、バンドルされたバイナリを介して`openclaw` CLIを公開します。
* 直接チャットはデフォルトでエージェントの`main`セッションに統合されます；グループは`agent:<agentId>:<channel>:group:<id>`として孤立したままです（ルーム/チャンネル: `agent:<agentId>:<channel>:channel:<id>`）；ハートビートはバックグラウンドタスクを生かします。

## コアスキル（設定 → スキルで有効化）

* **mcporter** — 外部スキルバックエンドを管理するためのツールサーバーランタイム/CLI。
* **Peekaboo** — オプションのAIビジョン分析を伴う高速macOSスクリーンショット。
* **camsnap** — RTSP/ONVIFセキュリティカメラからフレーム、クリップ、またはモーションアラートをキャプチャ。
* **oracle** — セッションリプレイとブラウザ制御を備えたOpenAI対応エージェントCLI。
* **eightctl** — ターミナルから睡眠を制御。
* **imsg** — iMessageおよびSMSを送信、読み取り、ストリーミング。
* **wacli** — WhatsApp CLI: 同期、検索、送信。
* **discord** — Discordアクション: 反応、ステッカー、投票。`user:<id>`または`channel:<id>`ターゲットを使用（生の数値IDは曖昧です）。
* **gog** — Google Suite CLI: Gmail、カレンダー、ドライブ、連絡先。
* **spotify-player** — 検索/キュー/再生制御のためのターミナルSpotifyクライアント。
* **sag** — macスタイルの音声UXを持つElevenLabs音声；デフォルトでスピーカーにストリーミング。
* **Sonos CLI** — スクリプトからSonosスピーカーを制御（発見/ステータス/再生/音量/グループ化）。
* **blucli** — スクリプトからBluOSプレーヤーを再生、グループ化、自動化。
* **OpenHue CLI** — シーンと自動化のためのPhilips Hue照明制御。
* **OpenAI Whisper** — クイックディクテーションとボイスメールのトランスクリプトのためのローカル音声認識。
* **Gemini CLI** — ターミナルからのGoogle Geminiモデルによる迅速なQ&A。
* **agent-tools** — 自動化とヘルパースクリプトのためのユーティリティツールキット。

## 使用上の注意

* スクリプティングには`openclaw` CLIを優先してください；macアプリは権限を処理します。
* スキルタブからインストールを実行してください；バイナリが既に存在する場合、ボタンは隠されます。
* ハートビートを有効に保ち、アシスタントがリマインダーをスケジュールし、受信トレイを監視し、カメラキャプチャをトリガーできるようにします。
* Canvas UIはネイティブオーバーレイでフルスクリーンで実行されます。重要なコントロールを左上/右上/下端に配置しないでください；レイアウトに明示的なガターを追加し、安全エリアのインセットに依存しないでください。
* ブラウザ駆動の検証には、OpenClaw管理のChromeプロファイルを使用して`openclaw browser`（タブ/ステータス/スクリーンショット）を使用してください。
* DOM検査には、`openclaw browser eval|query|dom|snapshot`を使用してください（および機械出力が必要な場合は`--json`/`--out`）。
* インタラクションには、`openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run`を使用してください（click/typeはスナップショット参照が必要です；CSSセレクタには`evaluate`を使用）。
