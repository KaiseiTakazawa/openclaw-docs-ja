# WebChat

# WebChat (macOSアプリ)

macOSメニューバーアプリは、WebChat UIをネイティブなSwiftUIビューとして埋め込んでいます。これはGatewayに接続し、選択されたAgentの**メインセッション**にデフォルトで接続します（他のセッション用のセッションスイッチャーがあります）。

* **ローカルモード**: ローカルGateway WebSocketに直接接続します。
* **リモートモード**: SSHを介してGateway制御ポートを転送し、そのトンネルをデータプレーンとして使用します。

## 起動とデバッグ

* 手動: Lobsterメニュー → “Open Chat”。

* テスト用の自動オープン:

  ```bash
  dist/OpenClaw.app/Contents/MacOS/OpenClaw --webchat
  ```

* ログ: `./scripts/clawlog.sh`（サブシステム `ai.openclaw`、カテゴリ `WebChatSwiftUI`）。

## 配線の仕組み

* データプレーン: Gateway WSメソッド `chat.history`、`chat.send`、`chat.abort`、`chat.inject` とイベント `chat`、`agent`、`presence`、`tick`、`health`。
* セッション: プライマリセッション（`main`、またはスコープがグローバルな場合は `global`）にデフォルト設定されています。UIはセッション間を切り替えることができます。
* オンボーディングは、初回設定を分離するために専用のセッションを使用します。

## セキュリティ面

* リモートモードは、Gateway WebSocket制御ポートのみをSSH経由で転送します。

## 知られている制限事項

* UIはチャットセッション向けに最適化されており（完全なブラウザサンドボックスではありません）。
