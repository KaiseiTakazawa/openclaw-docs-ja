## OpenClaw とは？

OpenClaw は **セルフホスト型ゲートウェイ** で、WhatsApp、Telegram、Discord、iMessage などのチャットアプリを AI エージェントに接続します。1つのゲートウェイプロセスを自分のマシン（またはサーバー）で実行し、メッセージングアプリと常時利用可能な AI アシスタントの橋渡しをします。

## クイックスタート

```bash
# OpenClaw をインストール
npm install -g openclaw@latest

# オンボード＆サービスインストール
openclaw onboard --install-daemon

# チャネル設定＆ゲートウェイ起動
openclaw channels login
openclaw gateway --port 18789
```
