# オーディオとボイスメモ

# Audio / Voice Notes — 2026-01-17

## What works

* **メディア理解（オーディオ）**: オーディオ理解が有効（または自動検出）されている場合、OpenClawは：
  1. 最初のオーディオ添付ファイル（ローカルパスまたはURL）を見つけ、必要に応じてダウンロードします。
  2. 各モデルエントリに送信する前に`maxBytes`を強制します。
  3. 最初の適格なモデルエントリを順番に実行します（プロバイダーまたはCLI）。
  4. 失敗したりスキップされた場合（サイズ/タイムアウト）、次のエントリを試みます。
  5. 成功した場合、`Body`を`[Audio]`ブロックに置き換え、`{{Transcript}}`を設定します。
* **コマンド解析**: 転写が成功すると、`CommandBody`/`RawBody`は転写された内容に設定されるため、スラッシュコマンドが引き続き機能します。
* **詳細なログ記録**: `--verbose`オプションを使用すると、転写が実行されるときとボディが置き換えられるときにログを記録します。

## 自動検出（デフォルト）

モデルを**構成しない**場合、かつ`tools.media.audio.enabled`が**false**に設定されていない場合、OpenClawは次の順序で自動検出を行い、最初の動作するオプションで停止します：

1. **ローカルCLI**（インストールされている場合）
   * `sherpa-onnx-offline`（`SHERPA_ONNX_MODEL_DIR`がエンコーダ/デコーダ/ジョイナー/トークンを必要とします）
   * `whisper-cli`（`whisper-cpp`から; `WHISPER_CPP_MODEL`またはバンドルされた小型モデルを使用）
   * `whisper`（Python CLI; モデルを自動的にダウンロード）
2. **Gemini CLI**（`gemini`）を使用して`read_many_files`
3. **プロバイダーキー**（OpenAI → Groq → Deepgram → Google）

自動検出を無効にするには、`tools.media.audio.enabled: false`を設定します。
カスタマイズするには、`tools.media.audio.models`を設定します。
注: バイナリ検出はmacOS/Linux/Windows全体で最善を尽くします; CLIが`PATH`にあることを確認してください（`~`を展開します）、または完全なコマンドパスで明示的なCLIモデルを設定します。

## 設定例

### プロバイダー + CLIフォールバック（OpenAI + Whisper CLI）

```json5
{
  tools: {
    media: {
      audio: {
        enabled: true,
        maxBytes: 20971520,
        models: [
          { provider: "openai", model: "gpt-4o-mini-transcribe" },
          {
            type: "cli",
            command: "whisper",
            args: ["--model", "base", "{{MediaPath}}"],
            timeoutSeconds: 45,
          },
        ],
      },
    },
  },
}
```

### プロバイダーのみでスコープゲーティング

```json5
{
  tools: {
    media: {
      audio: {
        enabled: true,
        scope: {
          default: "allow",
          rules: [{ action: "deny", match: { chatType: "group" } }],
        },
        models: [{ provider: "openai", model: "gpt-4o-mini-transcribe" }],
      },
    },
  },
}
```

### プロバイダーのみ（Deepgram）

```json5
{
  tools: {
    media: {
      audio: {
        enabled: true,
        models: [{ provider: "deepgram", model: "nova-3" }],
      },
    },
  },
}
```

### プロバイダーのみ（Mistral Voxtral）

```json5
{
  tools: {
    media: {
      audio: {
        enabled: true,
        models: [{ provider: "mistral", model: "voxtral-mini-latest" }],
      },
    },
  },
}
```

## ノートと制限

* プロバイダーの認証は標準のモデル認証順序（認証プロファイル、環境変数、`models.providers.*.apiKey`）に従います。
* Deepgramは`provider: "deepgram"`が使用されている場合に`DEEPGRAM_API_KEY`を取得します。
* Deepgramの設定詳細: [Deepgram (音声転写)](/providers/deepgram)。
* Mistralの設定詳細: [Mistral](/providers/mistral)。
* オーディオプロバイダーは`tools.media.audio`を介して`baseUrl`、`headers`、および`providerOptions`を上書きできます。
* デフォルトのサイズ制限は20MB（`tools.media.audio.maxBytes`）です。サイズオーバーのオーディオはそのモデルではスキップされ、次のエントリが試みられます。
* オーディオのデフォルト`maxChars`は**未設定**（完全な転写）。出力をトリムするには、`tools.media.audio.maxChars`またはエントリごとの`maxChars`を設定します。
* OpenAIの自動デフォルトは`gpt-4o-mini-transcribe`です; より高い精度のために`model: "gpt-4o-transcribe"`を設定します。
* 複数のボイスノートを処理するには`tools.media.audio.attachments`を使用します（`mode: "all"` + `maxAttachments`）。
* 転写はテンプレートに`{{Transcript}}`として利用可能です。
* CLIの標準出力は制限されています（5MB）；CLIの出力は簡潔に保ってください。

## グループ内のメンション検出

グループチャットに`requireMention: true`が設定されている場合、OpenClawはメンションを確認する前にオーディオを転写します。これにより、メンションを含むボイスノートも処理されるようになります。

**動作の仕組み:**

1. ボイスメッセージにテキストボディがなく、グループがメンションを必要とする場合、OpenClawは「プレフライト」転写を実行します。
2. 転写された内容がメンションパターン（例：`@BotName`、絵文字トリガー）をチェックします。
3. メンションが見つかった場合、メッセージは完全な返信パイプラインを通過します。
4. 転写はメンション検出に使用されるため、ボイスノートはメンションゲートを通過できます。

**フォールバック動作:**

* プレフライト中に転写が失敗した場合（タイムアウト、APIエラーなど）、メッセージはテキストのみのメンション検出に基づいて処理されます。
* これにより、混合メッセージ（テキスト + オーディオ）が誤ってドロップされることはありません。

**例:** ユーザーが「Hey @Claude, what's the weather?」と言うボイスノートを、`requireMention: true`のTelegramグループに送信します。このボイスノートは転写され、メンションが検出され、エージェントが返信します。

## 注意点

* スコープルールは最初に一致したものが勝ちます。`chatType`は`direct`、`group`、または`room`に正規化されます。
* CLIが0で終了し、プレーンテキストを出力することを確認してください; JSONは`jq -r .text`を介して整形する必要があります。
* タイムアウトは合理的に保ってください（`timeoutSeconds`、デフォルト60秒）; 返信キューをブロックしないようにします。
* プレフライト転写はメンション検出のために**最初の**オーディオ添付ファイルのみを処理します。追加のオーディオはメインのメディア理解フェーズ中に処理されます。
