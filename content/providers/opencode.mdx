# OpenCode Zen

OpenCode Zenは、コーディングエージェント向けにOpenCodeチームが推奨する**キュレーションされたモデルのリスト**です。
これは、APIキーと`opencode`プロバイダーを使用するオプションのホスティングモデルアクセスパスです。
Zenは現在ベータ版です。

## CLIセットアップ

```bash
openclaw onboard --auth-choice opencode-zen
# または非対話型
openclaw onboard --opencode-zen-api-key "$OPENCODE_API_KEY"
```

## 設定スニペット

```json5
{
  env: { OPENCODE_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "opencode/claude-opus-4-6" } } },
}
```

## 注意事項

* `OPENCODE_ZEN_API_KEY`もサポートされています。
* Zenにサインインし、請求情報を追加し、APIキーをコピーします。
* OpenCode Zenはリクエストごとに請求されます。詳細はOpenCodeダッシュボードを確認してください。
