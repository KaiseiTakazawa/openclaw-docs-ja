# Qianfan

# Qianfan プロバイダーガイド

QianfanはBaiduのMaaSプラットフォームで、**統一API**を提供し、単一のエンドポイントとAPIキーを介して多くのモデルへのリクエストをルーティングします。これはOpenAI互換であり、ほとんどのOpenAI SDKはベースURLを切り替えることで動作します。

## 前提条件

1. Qianfan APIアクセスのあるBaidu Cloudアカウント
2. QianfanコンソールからのAPIキー
3. システムにインストールされたOpenClaw

## APIキーの取得

1. [Qianfanコンソール](https://console.bce.baidu.com/qianfan/ais/console/apiKey)にアクセス
2. 新しいアプリケーションを作成するか、既存のものを選択
3. APIキーを生成（形式: `bce-v3/ALTAK-...`）
4. OpenClawで使用するためにAPIキーをコピー

## CLIのセットアップ

```bash
openclaw onboard --auth-choice qianfan-api-key
```
